# Smart Expense Tracker

A simple command-line expense tracking application written in C.

## Features

- Add expenses
- View all expenses
- Search expenses by category
- Calculate total spending
- Store data permanently using files

## Technologies Used

- C Programming
- File Handling
- Structures
- Functions

## Project Structure

smart-expense-tracker/

├── src/

├── data/

│
├── README.md

## How to Run

### Compile

gcc expense_tracker.c -o tracker

### Execute

./tracker
